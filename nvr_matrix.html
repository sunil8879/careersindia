Matrix Completion (2x2 & 3x3)

1. Theory: What is Matrix Completion?

Matrix Completion, also known as Figure Matrix, is a type of non-verbal reasoning problem where you are presented with a grid of figures, either 2x2 or 3x3, with one figure missing. The figures in the grid follow a specific pattern or rule, either horizontally (across rows) or vertically (down columns). Your task is to identify this pattern and select the correct figure from a set of options to complete the matrix.

How to Solve: A Systematic Approach

  1. Analyze the Matrix Structure: First, determine if the pattern progresses across the rows or down the columns. Sometimes, the pattern can apply to both, or even diagonally. Start by checking the rows.
  2. Row Analysis: Compare the first two figures in a complete row (e.g., row 1) to understand the change. See if the same change applies to the next complete row (e.g., row 2). If the logic is consistent, apply it to the incomplete row to find the missing figure.
  3. Column Analysis: If row analysis doesn't reveal a clear pattern, repeat the process for the columns. Compare the figures in a complete column to find the rule, and then apply that rule to the incomplete column.
  4. Identify the Pattern Type: Look for common transformations between figures:
    • Rotation: Figures rotate by a fixed degree (e.g., 45°, 90°, 180°) in a clockwise or anti-clockwise direction.
    • Addition/Subtraction: Elements like lines, dots, or shapes are added or removed in each step.
    • Superimposition/Combination: The third figure in a series is often a combination of the first two figures. Sometimes, common elements are removed (subtracted).
    • Movement: An element within the figure changes its position in a predictable sequence (e.g., moves from corner to corner).
    • Shading/Size: The shading of the figure changes, or the figure increases/decreases in size.
    • Mirror/Water Image: One figure is a reflection (mirror or water image) of another.
    • Analogy (for 2x2): The relationship between the top two figures is the same as the relationship between the bottom two. (A is to B as C is to ?).
  5. Predict and Select: Once you've identified the rule, apply it to find what the missing figure should look like. Then, carefully compare your prediction with the given options to find the exact match.
