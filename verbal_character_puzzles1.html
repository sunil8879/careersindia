CHARACTER PUZZLES

Theory: Introduction

Character puzzles are skill-based questions in which a figure or a matrix is given. Some numbers are filled in the matrix according to some rule. One space is left blank or marked with'?'. The candidate is required to crack the code or the rule through which the numbers are written. After applying that rule, he has to mark the correct option of the number which replaces '?' suitably.

There can be a number of rules through which character puzzles are formed. There can't be a single formula which applies to all questions. Cracking the character puzzles requires knowledge of squares, square roots, cubes, cube roots, prime numbers etc.

The more questions one practices, the more confidence one gains. Students come to know about the general rules used to frame the questions of character puzzles. Time factor is to be kept in mind while solving the character puzzles. Very often, students land up spending too much time on a single question.

Mastering these types of questions requires the use of basic operations of addition, subtraction, multiplication, division along with squares, cubes, prime numbers, square roots, multiples etc.
